# Variables d'environnement à remplir (.env)

## Application

| Variable | Description | Exemple |
|----------|-------------|---------|
| `APP_NAME` | Nom de l'application | `GrowFast` |
| `APP_KEY` | Clé de chiffrement (générée par `php artisan key:generate`) | `base64:xxx...` |
| `APP_URL` | URL de base (doit correspondre à l'URL d'accès pour OAuth) | `http://127.0.0.1:8000` ou `https://votredomaine.com` |

---

## Base de données (PostgreSQL)

| Variable | Description | Exemple |
|----------|-------------|---------|
| `DB_CONNECTION` | Type de BDD | `pgsql` |
| `DB_HOST` | Hôte PostgreSQL | `127.0.0.1` |
| `DB_PORT` | Port PostgreSQL | `5432` |
| `DB_DATABASE` | Nom de la base | `growfast` |
| `DB_USERNAME` | Utilisateur | `postgres` |
| `DB_PASSWORD` | Mot de passe | `votre_mot_de_passe` |

---

## JWT (Authentification)

| Variable | Description | Exemple |
|----------|-------------|---------|
| `JWT_SECRET` | Clé secrète JWT (générée par `php artisan jwt:secret`) | Chaîne aléatoire 64 caractères |
| `JWT_TTL` | Durée de vie du token en minutes | `60` |
| `JWT_REFRESH_TTL` | Fenêtre de refresh en minutes (2 semaines = 20160) | `20160` |
| `JWT_BLACKLIST_ENABLED` | Activer la révocation des tokens | `true` |

---

## Google OAuth

| Variable | Description | Où l'obtenir |
|----------|-------------|--------------|
| `GOOGLE_CLIENT_ID` | Client ID OAuth 2.0 | [Google Cloud Console](https://console.cloud.google.com/) → APIs & Services → Credentials |
| `GOOGLE_CLIENT_SECRET` | Client Secret OAuth 2.0 | Même emplacement |
| `GOOGLE_REDIRECT_URI` | URI de callback (optionnel si = APP_URL + `/api/auth/google/callback`) | `http://127.0.0.1:8000/api/auth/google/callback` |

**Important :** Ajouter l'URI exacte dans "Authorized redirect URIs" dans Google Cloud Console.

---

## LinkedIn OAuth

| Variable | Description | Où l'obtenir |
|----------|-------------|--------------|
| `LINKEDIN_CLIENT_ID` | Client ID | [LinkedIn Developers](https://www.linkedin.com/developers/) → Votre app → Auth |
| `LINKEDIN_CLIENT_SECRET` | Client Secret | Même emplacement |
| `LINKEDIN_REDIRECT_URI` | URI de callback (optionnel si = APP_URL + `/api/auth/linkedin/callback`) | `http://127.0.0.1:8000/api/auth/linkedin/callback` |

**Important :** Ajouter l'URI exacte dans "Authorized redirect URLs" dans LinkedIn Developer Portal.

---

## Notifications par email

| Variable | Description | Exemple |
|----------|-------------|---------|
| `NOTIFICATION_EMAIL` | Email qui reçoit les notifications à chaque étape (inscription, abonnement, scraping, matching) | `admin@growfast.com` |
| `NOTIFICATIONS_ENABLED` | Activer/désactiver les notifications | `true` |

---

## Gemini (recherche et extraction d'opportunités)

| Variable | Description | Exemple |
|----------|-------------|---------|
| `GEMINI_API_KEY` | Clé API Google AI (Gemini) | Obtenir sur [Google AI Studio](https://aistudio.google.com/apikey) |
| `GEMINI_MODEL` | Modèle (optionnel) | `gemini-2.0-flash` (défaut) ou `gemini-1.5-flash` |

`GEMINI_API_KEY` est requis pour la recherche d'opportunités via Gemini (bouton « Récupérer »).

---

## Commandes après modification du .env

```bash
php artisan key:generate
php artisan jwt:secret
php artisan config:clear
php artisan migrate
```
